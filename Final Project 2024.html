2) Machine Learning Analysis:

# Step 1: Preparing the Environment and Data
# Replace the path with the actual path to your downloaded dataset
d <- read.csv("cleaned_dataset_specific_columns.csv")

# Convert Runtime to numeric and ensure Released_Year is numeric
d$Runtime <- as.numeric(gsub(" min", "", d$Runtime), errors = "coerce")
d$Released_Year <- as.numeric(as.character(d$Released_Year), errors = "coerce")
## Warning: NAs introduced by coercion
# Remove NA values for a clean dataset
model_data <- na.omit(d[, c("IMDB_Rating", "Released_Year", "Runtime", "Meta_score", "No_of_Votes", "Gross")])
set.seed(123)  # For reproducibility
indices <- sample(1:nrow(model_data), size = 0.8 * nrow(model_data))  # 80% for training

# Step 2: Splitting the Data
train_data <- model_data[indices, ]
test_data <- model_data[-indices, ]

# Step 3: Training the Decision Tree Model
dt_model <- rpart(IMDB_Rating ~ ., data = train_data, method = "anova")

# Step 4: Visualizing the Decision Tree
rpart.plot(dt_model, type = 4, extra = 101)

# Step 5: Making Predictions and Evaluating the Model
predictions <- predict(dt_model, newdata = test_data, type = "vector")
actuals <- test_data$IMDB_Rating

# Calculate RMSE as an example of evaluation metric
rmse <- sqrt(mean((predictions - actuals)^2))
print(paste("RMSE:", rmse))
## [1] "RMSE: 0.20007284274144"

Interpretation:

Model Visualization: The decision tree model is then visualized. Each node in the tree represents a decision based on a condition on one of the variables. For example, one of the conditions in the tree is Meta_score < 53. This means that for movies with a Meta_score less than 53, the tree will follow the left branch, otherwise, it will follow the right branch. The tree continues to make decisions based on the variables until it reaches a leaf node, which contains the predicted IMDB_Rating.

Model Evaluation: The models predictions on the test data are compared to the actual IMDB_Rating values to calculate the Root Mean Square Error (RMSE). The RMSE is a measure of how well the models predictions match the actual values, with lower values indicating better performance.

If Meta_score < 53, check further conditions.

If Runtime < 125, predict liked.

If Meta_score >= 77, check No_of_Votes and Released_Year.

If No_of_Votes < 13,263 and Released_Year >= 1983, predict liked.

Otherwise, predict disliked.

The RMSE of 0.20 indicates the average prediction error in IMDB ratings.

The research question seems to be about predicting whether you will like a movie based on its IMDB_Rating. By using the decision tree, you can trust the model to guide your movie choices. The focus is on avoiding movies you dislike rather than missing potentially good ones.
